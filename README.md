# SQL-Driven Analysis of Data Engineer Job Market Trends
Analyzing Data Engineer job postings to uncover salary trends, in-demand skills, and hiring patterns using advance SQL techniques

## Project Overview
This project analyzes Data Engineer job postings to identify:
- Salary trends
- Most in-demand skills
- Remote work opportunities
- Geographic hiring patterns

## Tools Used
- SQL
- DuckDB
- VS Code
- GitHub

## Key SQL Concepts
- CTEs
- Window Functions
- Joins
- Subqueries
- Aggregations
